# Docker

## Перейти в папку с проектом

```aiignore

cd ToDoTT

```

## Запустить в Docker

```aiignore

docker-compose up --build

```

Swagger будет доступен по адресу: 

```aiignore

http://localhost:8080/swagger

```

# Локальный запуск

## Перейти в папку ToDo.API

```aiignore

cd ToDo.API

```

## Запустить

```aiignore

dotnet run

```

Swagger будет доступен по адресу:

```aiignore

http://localhost:5186/Swagger

```
